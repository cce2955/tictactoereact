# Tic Tac Toe Game

Welcome to my Tic Tac Toe game built using React! This interactive web application allows users to play the classic game of Tic Tac Toe against each other. It incorporates animations, grids, responsive input, and logic to provide an enjoyable gaming experience.

![React App - Brave 5_18_2023 2_53_16 PM](https://github.com/cce2955/tictactoereact/assets/44739551/2cc72ac4-9fc6-4558-b760-6e4ee59891ea)


## Features

- **Animations:** The game board and user interface are enhanced with smooth animations, providing a visually pleasing experience for players.

- **Grids:** The game board is designed as a 3x3 grid, representing the playing field. Players can click or tap on the grid cells to make their moves.

- **Responsive Input:** The user interface is responsive, adapting to different screen sizes and devices. Whether you're playing on a desktop, laptop, tablet, or mobile phone, the game will adjust accordingly.

- **Game Logic:** The game implements the logic of Tic Tac Toe, including win conditions, checking for a draw, and updating the game state. Players can enjoy a fair and competitive gaming experience.
